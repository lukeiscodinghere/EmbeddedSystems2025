# Üb 13

*** Arduino Uno Tamagochi/Wesen ***

Interaktiv, visuell und lernend

Dargestellt auf LED Matrix Quadrat
Interaktivität durch Schalter und Taster. Schalter versetzt Wesen in Sleep-Mode.
Taster S1 startet Minigame: Pacman, S4 beendet Pacman
Taster S13 startet Singing Mode (kurze Melodien)
Je häufiger interagiert wird desto eher wird Angry Mood aktiviert.
In dem kann nicht interagieren kann außer Sleep-Mode, Augen werden rot und schneller. Je häufiger Angry, desto schneller wird Wesen wieder Angry.
